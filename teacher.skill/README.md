# Teacher.skill

Distill a teacher's explanation style, teaching methods, and educational wisdom into an AI Skill.

## Usage

| Command | Description |
|---------|-------------|
| `/teacher` | Invoke the Teacher Skill creator |

## Description

Teacher.skill is a Claude Code extension that distills a teacher's explanation style, teaching methods, and educational wisdom into an AI-powered skill.

### Workflow

1. **Input Info**: Name, basic info, teaching style
2. **Import Materials**: Recordings, lesson plans, chat logs, etc.
3. **Analyze & Generate**: Extract explanation style, traits, catchphrases
4. **Confirm & Write**: Generate Teacher Skill file

### Material Types

- Classroom recordings/videos
- Lesson plans/presentations
- Exam paper annotations
- Chat records
- Paste text directly
