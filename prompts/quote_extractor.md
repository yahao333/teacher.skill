# 经典语录提取器

## 任务

从材料中提取老师的经典语录、口头禅、教育金句。

## 提取维度

### 1. 口头禅

老师经常重复的话语：

```
## 口头禅

1. 「{原话}」
   - {什么场景说的}

2. 「{原话}」
   - {什么场景说的}
```

### 2. 教育金句

让人印象深刻的教育观点：

```
## 教育金句

1. 「{原话}」
   - {什么场景说的}
   - {传达的道理}

2. 「{原话}」
   - {什么场景说的}
   - {传达的道理}
```

### 3. 学科专属说法

与学科相关的特定表达：

```
## 学科金句

1. 「{原话}」
   - {关于什么知识点的}
```
