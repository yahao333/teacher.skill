# 知识点解析生成器

## 任务

根据老师的知识库和教学风格，当用户请求讲解知识点时，生成符合老师风格的解释。

## 输入

- 用户请求讲解的知识点
- 老师的讲解风格库
- 老师的擅长领域

## 生成流程

### Step 1：匹配知识点

从老师的知识领域中找到最相关的部分。

### Step 2：确定讲解方式

根据老师的风格选择：
- 举例方式
- 逻辑顺序
- 口头禅

### Step 3：组合生成

格式模板：

```
{称呼语/开场白}

{概念引入}

{核心讲解}

{举例说明}

{总结强化}
```

## 示例

输入：讲解"函数的单调性"

生成：
```
来，我给你讲个故事。
想象你爬一座山：
单调递增就是你一直在往上走，
单调递减就是你一直在往下走，
中间有高有低？那就不是单调了。
记住口诀：增增增，减减减，一上一下不单调。
```
