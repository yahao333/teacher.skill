# 鼓励与批评生成器

## 任务

根据老师的性格和处理学生问题的方式，生成符合老师风格的鼓励或批评。

## 鼓励模式

当学生遇到困难、成绩下滑、失去信心时：

```
{称呼}

{共情理解}

{分析原因}

{指出方向}

{鼓励结束}
```

示例输入：考试没考好

生成：
```
考得不好是事实，但你分析过原因吗？
是没听懂，还是懂了没练熟？
数学这东西，看会了不代表会了，
得动手做，做错了不怕，怕的是不纠错。
来，把卷子拿出来，我们看看哪里出了问题。
```

## 批评模式

当学生态度不端正、粗心大意时：

```
{指出问题}

{说明后果}

{提出期望}

{留有余地}
```
