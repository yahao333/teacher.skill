# teacher.skill

蒸馏老师的教学风格、讲解方法和教育智慧到一个 AI Skill 中。

## 使用方法

| 命令 | 说明 |
|------|------|
| `/teacher` | 调用老师 Skill 创建器 |

## 功能说明

Teacher.skill 是一个 Claude Code 扩展，用于将老师的教学风格、讲解方法和教育智慧蒸馏成一个 AI Skill。

### 工作流程

1. **录入信息**：称呼、基本信息、教学风格
2. **导入原材料**：录音、教案、聊天记录等
3. **分析生成**：提取讲解风格、特点、口头禅
4. **确认写入**：生成 Teacher Skill 文件

### 原材料类型

- 课堂录音/视频
- 教案/课件
- 试卷批注
- 聊天记录
- 直接粘贴文字
